TO LOAD THE DATABASE IN GOOGLE COLLAB

1. go to "Shared with me" inside your google drive

2. right-click on the folder "Gradient_Explorers_shared_folder"

3. click on "organize"

4. click on "add shortcut" 

5. select "my drive" as folder

a new shortcut will be generate in "my drive" folder ready to be loaded

FOR THE HUGGING FACE SECRET

1. create a new token in your huggingface profile

2. go to the left part of collab in the "secret" section. It has an icon of an old key

3. add secret

4. in the column of "name" write "HF_TOKEN"

5. in the column of Value add the  token value from huggingface
